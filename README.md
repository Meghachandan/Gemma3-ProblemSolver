# AI Math Problem Solver & Knowledge Assistant

## Overview

AI Math Problem Solver & Knowledge Assistant is an intelligent web application built with **Streamlit**, **LangChain**, and **Groq LLMs**. It is designed to solve mathematical problems, explain logical reasoning step by step, and answer general knowledge questions using Wikipedia as an additional information source.

Instead of only providing final answers, the application focuses on delivering clear explanations that help users understand the reasoning behind each solution.

---

## Features

* Solve arithmetic, algebra, and mathematical word problems.
* Generate detailed step-by-step solutions.
* Perform logical and analytical reasoning.
* Retrieve relevant information from Wikipedia for general knowledge questions.
* Natural language question answering.
* Fast responses powered by Groq's Llama 3.3 model.
* Clean and interactive Streamlit interface.
* Conversation history during the current session.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Move into the project directory:

```bash
cd your-repository
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Application

Start the Streamlit application:

```bash
streamlit run app.py
```

---

## Usage

1. Launch the application.
2. Enter your Groq API key in the sidebar.
3. Type a mathematical, logical, or general knowledge question.
4. Click **Find My Answer**.
5. Review the generated explanation and final answer.

## Example Questions

* Solve: (25 × 14) + 96 ÷ 4
